# xLaDe Demo Project

A minimal demo showing how to run an experiment under xLaDe.

Usage:
```bash
# from repo root
./bin/xlade experimental ./experiments/exp-001-proof-review/run.sh
```

This demo intentionally uses the in-repo experiments.


**Commands:**
```bash
mkdir -p projects/demo
touch projects/demo/README.md
# add text to README.md
git add projects/demo/README.md
git commit -m "Add simple xLaDe demo project"
git push
```
