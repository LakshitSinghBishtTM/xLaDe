# Installing xLaDe

xLaDe is a Lean-based ecosystem framework (distribution + experiments).

## Requirements
- Git
- Lean 4 toolchain (https://leanprover.github.io/)
- Lake build tool (comes with Lean 4 toolchain)

## Quick install
```bash
git clone --recurse-submodules https://github.com/LakshitSinghBishtTM/xLaDe.git
cd xLaDe
# optionally make the CLI executable
chmod +x bin/xlade
