# Onboarding Mode

## Purpose
Onboarding mode provides a minimal and forgiving Lean experience
for new contributors.

## Characteristics
- Builds only curated examples
- Emits helpful messages
- Avoids strict policy enforcement
- Optimized for learning and exploration

## Intended Users
- First-time Lean users
- New xLaDe contributors
- Educational contexts

## Stability
Non-stable. Intended for ease of entry, not guarantees.
